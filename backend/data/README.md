# Datasets for BloodIQ ML Models

Please place the downloaded datasets in this directory (`backend/data/`).

## 1. Demand Dataset (Kaggle)
* **Source:** https://www.kaggle.com/datasets/rishi2003das/blood-demand-dataset
* **Instructions:** Download the dataset from Kaggle, extract the CSV, and place it here.
* **Expected Filename:** `blood_demand.csv`

## 2. Supply Dataset (Data.gov.my)
* **Source:** https://data.gov.my/data-catalogue/blood_donations
* **Instructions:** Download the CSV for daily blood donations.
* **Expected Filename:** `blood_donations.csv`

Once both files are present, we can run the ML training scripts!
